← Launch Hub

Daily Workflow

  1. Review intake queue.
  2. Classify document or request type.
  3. Route to vertical lane.
  4. Run BNCA.
  5. Assign owner lane.
  6. Update client status.
  7. Escalate unresolved risk to Strategist.

Weekly Client Report

  1. Summarize completed work.
  2. List unresolved blockers.
  3. Show KPI movement.
  4. Provide next-week BNCA.
  5. Identify automation opportunities.

Ameris PoC SOP

  1. Collect 5–10 docs.
  2. Process each through construction suite.
  3. Extract missing info, risk, timeline, approvals.
  4. Generate next actions.
  5. Present custom Ameris portal.
